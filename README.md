# java-web-api-gradle-spark-hello-world

## Description
A POC for java spark webframework.

## Tech stack
- java
- gradle
  - spark

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`
- Available at http://localhost

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
